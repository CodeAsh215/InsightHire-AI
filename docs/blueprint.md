# **App Name**: InsightHire AI

## Core Features:

- AI Interview Simulation: Real-time AI interviewer that acts as a tool to adapt questions dynamically based on user responses (via voice or text), providing live sentiment and confidence detection.
- Specialized Interview Modes: Select from targeted interview modes including Data Structures & Algorithms (DSA), Core CS topics, HR Behavioral, and System Design.
- AI Resume-Based Questioning: Utilize an AI tool to generate personalized interview questions by analyzing the user's uploaded resume for highly relevant queries.
- AI Performance Analytics Dashboard: A comprehensive dashboard that uses an AI tool to provide feedback on technical accuracy, communication clarity, confidence level, filler word count, weak topic detection, and interview progress. Users can also download a PDF performance report.
- User Authentication & Profile: Secure user registration and login functionality, including Google and LinkedIn SSO, for profile management and progress tracking.
- Freemium Subscription Management: Implement a freemium model with distinct feature access (e.g., number of interviews, advanced analytics) for free and Pro tier users, with payment integration for upgrades.
- Gamification & Social Sharing: Engage users with skill badges, streak systems, leaderboards, and shareable result cards for platforms like LinkedIn.

## Style Guidelines:

- Color scheme is dark-mode first. The primary color, hinting at intelligence and sophistication, is a deep violet (#824DBA). The background features a very dark, desaturated violet hue (#231E27) for a premium, clean foundation. An accent color, a brighter, analogous light blue (#B4CDFC), is used sparingly for highlights and interactive elements, ensuring strong contrast and a futuristic feel.
- Primary and body font: 'Space Grotesk', a sans-serif typeface that delivers a computerized, techy, and modern scientific feel, aligning with the AI-powered and futuristic brand identity. Its clear legibility is suitable for both headlines and short passages of body text common in an interactive interview simulator.
- Utilize minimalist, modern icons with subtle gradients or glass-like effects that complement the dark-mode and futuristic aesthetic, avoiding overly complex or skeuomorphic designs. Icons should be easily decipherable and convey their meaning instantly.
- A distraction-free, highly structured layout with premium spacing and generous padding, inspired by the clean aesthetics of Stripe and Linear. Implement glass UI panels with rounded corners to create a subtle depth effect over soft, blurred gradients, giving a sense of high-end transparency and polish.
- Incorporate smooth transitions, micro-animations, and subtle glowing accents throughout the UI. Interactions such as hover states, element loading, and navigation changes should be fluid and elegant, enhancing the premium feel without being distracting, as inspired by Framer Motion.
